How {{ getTeachingHours() }} is calculated for {{ report.teacher.name }}

Calculation

Teaching time is the sum of scheduled class durations for all past meetings assigned to this teacher.

  • If a meeting has a recorded duration, that value is used.
  • If attendance was recorded but duration is missing, 60 minutes is estimated per meeting.
  • Meetings without attendance or duration contribute 0 minutes.
