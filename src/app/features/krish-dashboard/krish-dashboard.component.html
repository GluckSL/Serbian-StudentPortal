Import Students

Upload your CRM export (CSV or Excel). These column names are supported automatically:

  • Candidate Name → name
  • Email Address → email
  • Package Opted → Platinum / Silver / Visa & Docs
  • Current Status → Not Started / Uncertain / Ongoing / Completed / Withdrew
  • Phone Number, WhatsApp Number, Age
  • Assigned Sales Representative → counselor
  • Service Opted → CV Service, Interview Preparation, etc.
  • Professional or Professional Categories → IT Professional, Nurse, Engineer, O/L, etc.
  • Specialization → stored separately, not used for profession filters

Short aliases also work: name, email, package, status.
